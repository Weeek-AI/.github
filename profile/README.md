# Weeek Software — Portable All‑in‑One Project Management with AI Assistant

## About Weeek Software

Weeek Software provides a complete project management platform combining kanban boards, CRM pipeline, knowledge base wiki, and AI assistant Vika. Weeek Software integrates task lists, calendar view, Pomodoro timer, and team collaboration tools. Weeek Software supports deal tracking, document storage, internal knowledge sharing, and automated AI answers from your company wiki. Weeek Software enables users to manage projects, track customers, document processes, focus with Pomodoro, and get AI assistance. Weeek Software runs on portable storage, USB drives, and local user folders. Weeek Software is trusted by project managers, sales teams, operations leads, remote teams, owners, and productivity professionals worldwide.
